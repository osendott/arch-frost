Arch-Frost is a custom Arch-blue Numix-Frost GTK theme with a new, custom Gnome-Shell theme to match.

Arch-Frost comes in both a light and dark version

You can download the lastest version by downloading the arch-frost.zip or arch-frost.tar.gz archives above, or if you want to see what's being changed prior to the next release, you can clone the repo https://github.com/osendott/arch-frost.git and run the installer to copy the files to your ~./themes folder.(if you need to place the files elsewhere, edit the install script or just move them manually.) You could also symlink the folders to your ~/.themes folder, that way each time you pull an update from the repo, it'll auto-update your theme without you having to keep moving the folders. Just remember you will be using a testing version and there could be errors. More than likely things will be ok I do not push anything until it has been tested, however sometimes things slip through.

If you have any comments or suggestions, feel free to e-mail me at the following address: william.osendott[at]gmail[dot]com or if you're a GitHub user, clone the repo, make your changes, and submit a pull request.
