Arch-Frost

Arch-Frost is a custom Arch-blue Numix-Frost GTK theme with a whole new, custom gnome-shell theme to match.

You can download get the lastest version by downloading the arch-frost.zip or arch-frost.tar.gz archives above, or if you want to see what's being changed prior to the next release, you can clone the repo https://github.com/osendott/arch-frost.git and symlink or copy the appropriate folders to your "~/.themes" directory (or wherever your distro stores theme files), but beware you will be using an untested version and there could be errors. More than likely you will experience no problems as I do not push anything until it has been tested, however if you wish to download the latest stable version, download the arch-frost.tar.gz or arch-frost.zip file from the main repo.

If you have any comments or suggestions, feel free to e-mail me at the following address: william.osendott[at]gmail[dot]com or if you're a GitHub user, clone the repo, make your changes, and submit a pull request.

If you experience any problems, please raise the issue on the GitHub page: https://github.com/osendott/arch-frost/issues
