Arch-Frost

Arch-Frost is a custom Arch-blue Numix-Frost GTK theme with a whole new, custom gnome-shell theme to match.

Arch-Frost comes in both a light and dark version

You can download get the lastest version by downloading the arch-frost.zip or arch-frost.tar.gz archives above, or if you want to see what's being changed prior to the next release, you can clone the repo https://github.com/osendott/arch-frost.git and run the installer to copy the files to you ~./themes folder (if you need them elsewhere, edit the install script or move them manually), but rmember you will be using a testing version and there could be errors. More than likely things will be ok I do not push anything until it has been tested, however sometimes things slip through.

If you have any comments or suggestions, feel free to e-mail me at the following address: william.osendott[at]gmail[dot]com or if you're a GitHub user, clone the repo, make your changes, and submit a pull request.

If you experience any problems, please raise the issue on the GitHub page: https://github.com/osendott/arch-frost/issues
